# Network Scanning Tools

## Description

This project automates network scanning using Python. It includes Ping Scanner, ARP Scanner, and Nmap Scanner.

## Requirements

* Python 3
* Nmap installed

## How to Run

### Ping Scanner

python ping\_scanner.py

### ARP Scanner

python arp\_scanner.py

### Nmap Scanner

python nmap\_scanner.py

## Example

Enter IP like:
127.0.0.1
