# Network Scanning Tools

## Description
This project automates network scanning using Python. It includes Ping Scanner, ARP Scanner, and Nmap Scanner.

## Requirements
- Python 3
- Nmap installed

## How to Run

### Ping Scanner
python ping_scanner.py

### ARP Scanner
python arp_scanner.py

### Nmap Scanner
python nmap_scanner.py

## Example
Enter IP like:
127.0.0.1
